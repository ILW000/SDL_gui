# SDL_gui
GUI library for SDL2

## This is work in progress!

## Features
- Base on SDL, SDL_image, SDL_ttf
- Support HarffBuzz to display international text correctly.
- Written in C++

## Multi-platform
- MacOS
- Windows
- iOS
- Android
- WebAssembly (via Emscripten)
- Raspberry Pi
- Linux (Build with CMake)
